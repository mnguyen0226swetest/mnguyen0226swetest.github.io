# mnguyen0226swetest-phdhub

This project is a GitHub Pages website for a non-profit organization, designed with a modern tech company theme. It includes essential pages such as Home, About Us, and News, providing a platform to share the organization's mission, updates, and events.

## Features
- Home Page: The main entry point with navigation links to other sections.
- About Us Page: Details the mission, vision, and values of the organization.
- News Page: Updates, events, and announcements related to the organization.
- Modern Tech Company Theme: A clean and responsive design that enhances user experience.

## Project Structure
```
mnguyen0226swetest-phdhub
├── index.html        # Home Page
├── about.html        # About Us Page
├── news.html         # News Page
├── css
│   └── style.css     # Styles for the website
├── js
│   └── main.js       # JavaScript functionality
├── assets
│   └── .gitkeep      # Placeholder for media files
└── README.md         # Project documentation
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Open `index.html` in a web browser to view the Home Page.
3. Navigate to the About Us and News pages using the provided links.
